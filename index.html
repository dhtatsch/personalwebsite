Sobre

Olá! Me chamo Douglas e tenho formação em Engenharia de Computação pela Universidade de Santa Cruz do Sul. Trabalho na área desde 2014, onde comecei estagiando e me tornei desenvolvedor de software. Atualmente trabalho com desenvolvimento Web em C# utilizando framework ASP NET MVC, juntamente com JavaScript (JQuery), CSS, Bootstrap, Dapper, SQL Server e HTML.

Dentro das minhas atividades também estão o levantamento de requisitos junto aos clientes e modelagem de banco de dados em SQL Server. Também atuo no desenvolvimento de aplicativos mobile para IOS e Android no framework TotalCross utilizando Java como linguagem de programação e SQLite como base de dados.

Além disso também faço a organização de tarefas e gerenciamento de produto utilizando metodologia SCRUM e a ferramenta Jira no meu dia a dia. Também auxílio o time com dúvidas técnicas e regras de negócio propondo as melhores soluções dentro do possível.
